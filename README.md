# 🏃‍♂️ Run Forever

**Run Forever** is a modern fitness tracking app designed to help you stay active, motivated, and consistent. It tracks your daily steps, running distance, calories burned, and activity time — all wrapped in a smooth, minimal, Apple-inspired UI.

---

## ✨ Features

- 👟 **Step Tracking** – Monitor your daily steps in real-time  
- 📏 **Distance Calculation** – Track how far you’ve run or walked  
- 🔥 **Calories Burned** – Stay aware of your fitness progress  
- ⏱️ **Run Timer** – Measure duration of your activities  
- 📊 **Activity History** – View past runs with clean card layouts  
- 📈 **Analytics Dashboard** – Weekly & monthly performance insights  
- 🎯 **Goal Setting** – Set and achieve daily fitness targets  
- 🌙 **Dark Mode Support** – Easy on the eyes anytime  
- ⚡ **Smooth UI** – Fluid animations and premium user experience  

---

## 🎨 Design Philosophy

Run Forever follows a **minimal, elegant, and user-first design**, inspired by Apple’s Human Interface Guidelines:

- Clean layouts with plenty of whitespace  
- Soft gradients and subtle shadows  
- Rounded corners and modern typography  
- Smooth transitions and micro-interactions

## 🛠️ Installation

```bash

git clone https://github.com/arjuniverse/RunForever.git

# Go into the project folder
cd RunForever

# Install dependencies
npm install

# Run the app
npm start
```
